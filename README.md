# Zen Tab Loading

A mod for **Zen Browser** that transforms the tab loading indicator into a modern spinner with glow, fluid rotation, and a smooth pulse in the title.

### Resources
- Spinner with glass effect + radial glow
- Rotation animation with slight scaling.
- Elegant pulse in the tab text
- Compatible with light and dark themes.
- It works best with `ui.prefersReducedMotion` = 0 in about:config

### Manual Installation
1. Open Zen, `about:support`, **Open Folder** (Profile Directory)
2. Create the `chrome` folder (if it doesn't already exist).
3. Create the file `userChrome.css` and past the content of chrome.css file inside it.
4. Restart Zen Browser
